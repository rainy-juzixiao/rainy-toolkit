# core命名空间公开API

本文档介绍了核心库中，面向用户的公开API部分。

其中，core主命名空间定义了以下的类

| 类                                      | 说明                    |
| -------------------------------------- | --------------------- |
| [basic_any](/md/core/public_api_core/basic_poly) | 一个基础的多态包装器，用于避免虚表的开销。 |

另外，core主命名空间也定义了如下的函数

| 函数                                               | 描述                    |
| ------------------------------------------------ | --------------------- |
| [accumulate](/md/core/public_api_core/accmulate) | 计算给定范围内所有元素与初始值的累计结果。 |
