# 欢迎来到 rainy-toolkit meta库项目文档

meta库是 rainy-toolkit 的一个重要组成部分，旨在提供高级的元编程工具和类型操作功能，帮助开发者更高效地进行 C++ 编程。

meta库允许你反射和操作类型信息，支持复杂的类型转换和多态行为。它包含了一系列模板类和函数。meta主要分为两大模块：

- moon
- lunar

其中，moon模块是专注于静态的元编程。moon通过模板元编程技术获取类的信息，以执行编译期可确定的元编程操作。

对于moon模块的使用，你可以参见moon库的快速导览。此处，仅作简单介绍。moon允许你借助`enumeration`库，你可以轻松的对枚举类型进行反射，获取枚举成员名称和值的映射关系。以及，也可以使用`tuple_like`库，直接获取聚合类型的成员信息，同时可以根据需要，获得private成员的信息以及成员偏移量。总而言之，moon模块为可以允许用户进行相对强大的编译期类型操作和反射。同时，moon模块也被作为lunar的模块基础进行构建，同时也被用于实现`basic_any`等涉及高级操作的功能。

而lunar模块，则相对专注于动态性。例如，lunar模块提供了反射库，也相应的，提供了更安全，封装更好的`poly`库。并且，专注于对类型的高度泛化。

在本文档中，你可以直接根据需要，查阅左侧导航栏中的各个模块和功能的详细说明和使用示例。我们提供了示例，也提供了API参考，让你能更好的理解meta库的机制。
