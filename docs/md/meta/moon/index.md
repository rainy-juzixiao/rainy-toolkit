# 欢迎来到rainy-toolkit meta库 moon模块项目文档

moon模块是专注于静态的元编程。moon通过模板元编程技术获取类的信息，以执行编译期可确定的元编程操作。

对于moon模块的使用，你可以参见moon库的快速导览。

总而言之，moon提供了两个头文件用于帮助你进行元编程。

`<tuple_like.hpp>`:

可以将一个聚合类型，直接变为直接获取聚合类型的成员信息，同时可以根据需要，获得private成员的信息以及成员偏移量。例如，`tuple_like`可以将聚合类型看作元组，允许直接操作

`<enumeration.hpp>`:

可以将一个枚举类型进行反射，获取枚举成员名称和值的映射关系。以及，也可以使用`tuple_like`库，直接获取聚合类型的成员信息，同时可以根据需要，获得private成员的信息以及成员偏移量。

此外，moon模块为可以允许用户进行相对强大的编译期类型操作和反射。同时，moon模块也被作为lunar的模块基础进行构建，同时也被用于实现`basic_any`等涉及高级操作的功能。
